"WP Awesome Insta Widget" is A very Simple but Extremely Powerful Instagram Widget Plugins has lots of Built in Options like - Control width of Columns, Image size control etc.

If you are Searching for the right plugin to display your latest Instagram photos and videos in your sidebar? "WP Awesome Insta Widget" Plugin comes with a lot of possibilities to add your Photos or Videos in your WordPress Website Sidebar or Footer. You can customize your Photos height-width with the column. This Plugin compatible with any WordPress Theme. 


= Features:  =
* Customize Hight and Width
* Show/Hide Likes and Comments on Hover
* Works on all Theme
* Show/Hide Follow Me 
* Open New Tab or Same Tab
* Different Column View

The Installaiton Process is very simple

e.g.

1. Upload 'wp-awesome-insta-widget.zip' file to Plugin Uploader or
2. Upload 'wp-awesome-insta-widget' to the '/wp-content/plugins/'' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
